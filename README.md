# 🎵 Chinook Database Analysis (SQL + Tableau)

## 📊 Overview
This project analyzes a digital music store database (Chinook) using SQL and Tableau to uncover customer behavior, revenue trends, and employee support.

## 🎯 Objectives
- Identify top customers by total spending
- Analyze revenue distribution across countries
- Determine best-selling artists and genres
- Evaluate employee support performance

## 🔧 Tools Used
- PostgreSQL (pgAdmin)
- SQL
- Tableau

## 📈 Key Analysis
- Customer spending analysis
- Revenue by country
- Top-selling artists
- Genre popularity
- Employee workload analysis

## 📁 Files
- `chinook_analysis.sql` → SQL queries
- `chinook_dashboard.twbx` → Tableau File
- `top_customers.png` → Tableau Visualization
- `most_sold_genres.png` → Tableau Visualization
- `revenue_by_country.png` → Tableau Visualization
- `revenue_by_artist.png` → Tableau Visualization
- `employee_support.png` → Tableau Visualization

## 💡 Key Insights
- A small number of customers generate a large portion of revenue
- Certain countries dominate sales 
- Specific genres and artists consistently perform better
